# 🆘 RESPONDER — Emergency Response System
Hackathon Project · May 2026

---

## 🚀 Launch
1. Extract `RESPONDER` folder to Desktop
2. Open PowerShell inside → `Shift + Right Click` → "Open PowerShell here"
3. Run: `python server.py`
4. Open: `localhost:8000/RESPONDER.html`

---

## 📱 Features

### 📋 Disaster Guides
- **Offline:** Flood, Fire, Earthquake, Chemical leak, Go-Bag checklist, Evacuation routes
- **AI Guide:** Describe your situation → get personalized step-by-step instructions

### 🩺 Medicine
- **Offline:** First Aid (CPR/bleeding/burns), Vital Signs, Triage, Tourniquet Timer, Hypothermia, Medical Card
- **AI Advisor:** Text or photo → AI identifies severity, injury type, poison risk, treatment steps

### 🌿 Plant ID *(online)*
Photo → AI identifies plant/mushroom/berry → EDIBLE / CAUTION / DO NOT EAT

### 🆘 Emergency Rescue
- 📍 **GPS Position** — coordinates + SMS with one tap
- 🔊 **SOS Alarm** — max volume SOS Morse for 60 seconds + strobe light
- 🛡️ **Dead Man's Switch** — set timer (5-120 min), check in regularly. If no response → auto email with GPS to emergency contact
- 📻 MAYDAY Script · 🪞 Ground Signals · 📡 Emergency Frequencies
